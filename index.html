I'm a french developer, musician and ~graphist.

As a developer I mainly do Graphics and Audio DSP programming using the C++ language, OpenGL and raw buffer manipulation (that's my current full-time job). I'm also ok with web development, Python scripting, and generally make anything (ie C program) work anywhere (ie web, mobile devices, etc.).

I'm also experimenting with shaders and other "artistic" compositions with code, mainly using GLSL, THREE.js, html5 canvas or whatever comes to mind

As a musician I do live sets and compositions using an eurorack modular synth, but also computer stuff, with Ableton and the usual suspects.

As a graphist I mainly do the visuals for my other projects (live gigs, applications). I did draw a lot more in the past (pen & paper) but not so much now.
